# InnoShop

Проект "Inno_Shop" представляет собой систему, состоящую из двух микросервисов, разработанных на ASP.NET Core, предназначенных для управления пользователями и их продуктами.

# Микросервис №1: Users.Api (Управление Пользователями)
Этот микросервис отвечает за все, что связано с жизненным циклом учетных записей.

## Основные возможности
- Полный CRUD для сущности User (создание, чтение, обновление, удаление)
- Аутентификация с использованием JWT
- Авторизация на основе ролей (User, Admin)
- Хеширование паролей (IPasswordHasher)
- Подтверждение аккаунта через Email (отправка токена)
- Восстановление пароля (генерация токена сброса)
- Управление активностью: активация/деактивация пользователей (функция администрирования)
- Межсервисное взаимодействие: инициирование скрытия продуктов пользователя в Products.Api при его деактивации

# Микросервис №2: Products.Api (Управление Продуктами)
Этот микросервис отвечает за хранение и управление информацией о товарах.

## Основные возможности
- Полный CRUD для сущности Product (создание, чтение, обновление, удаление)
- Авторизованные операции: добавление, удаление и редактирование доступны только авторизованным пользователям и только для их собственных продуктов
- Функционал поиска и фильтрации по разным параметрам
- Механизм SoftDelete: продукты скрыты при деактивации их создателя в Users.Api и автоматически восстановлены при активации

## Используемые технологии
- C# 12 / .NET 8 / ASP.NET Core Web API
- Базы данных	MSSQL Server / Entity Framework Core
- Архитектура	MediatR / Clean Architecture / CQRS
- JWT (JSON Web Tokens) / IPasswordHasher
- Валидация	FluentValidation
- Swagger
- Тестирование	xUnit, Moq - unit и интеграционное тестирование
- Развертывание	Docker / Docker Compose
